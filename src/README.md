
# Frontend

This directory contains the frontend code for the chatbot application.

## Files

- `src/theme/Chatbot.tsx`: This is the main component for the chatbot. It handles the chat interface, sending messages to the backend, and displaying the responses.
- `src/css/chatbot.css`: This file contains the styles for the chatbot component.
- `src/pages/index.tsx`: This is the main page of the application. It includes the `Chatbot` component.
- `src/theme/Layout.tsx`: This is the layout for the application. It includes the `Chatbot` component.

## Running the Frontend

To run the frontend, you need to have Node.js and npm installed.

1.  Navigate to the root directory of the project.
2.  Install the dependencies: `npm install`
3.  Start the development server: `npm start`

This will start the frontend application on `http://localhost:3000`.
